Support

NASTY is a free family board game with no account to manage, so most "support" is just a quick question. Email blake.pangman@gmail.com and you'll hear back directly.

Common questions

  • My game disappeared / the app restarted. Reopen it — offline games auto-save and offer a "Resume your game" button on the main menu. Online games are saved on the server too, so reopening the same invite link (or just reopening the app) reconnects you to the same table.
  • Online play isn't connecting. Online games need the host's game server to be reachable. This is a small family-run server, not a big company's cloud — if it's briefly offline (power blip, restart), just try again in a minute.
  • How do the rules work? Tap "Rules" from the main menu inside the app any time — the full card-by-card rundown is right there.
  • I found a bug or have an idea. Email the address above — feedback from the family is exactly how this app gets better.

Learn more about the game at bpangman.github.io/nasty.
